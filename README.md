## ✨ 功能

- 🎭 **选心情** — 从 5 种情绪里选一个，塔罗要看菜下碟
- 🃏 **抽牌** — 随机抽取 3 张塔罗牌
- 🍜 **看结果** — 根据牌面给出今日美食推荐 + 幽默解读

## 🚀 本地运行

```bash
# 任意 HTTP 服务器均可，例如：
python3 -m http.server 3456
# 然后打开 http://localhost:3456
```
